# Video Chat Web Application

This is a a peer-to-peer video chat web app. You can copy the room's link and share it with anyone you want to meet. 

## Technologies
* Node.js
* Express
* Peer.js

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```
Run the app

```sh
$ node app.js
$ node peerjs --port 3001
```
